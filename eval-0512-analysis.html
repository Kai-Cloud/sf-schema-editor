Q103 (M8 — Quote) — Pass → Fail

  • Query: What is the discount percentage on the most recently created GenePoint quote?
  • Why it passed without M8: Old assertion was "GenePoint Lab Q2 Quote (Discount 8.0%)" — the assertion only checked whether "8.0%" appeared anywhere in the response. The Quote object is not crawled by OOB, so the assistant most likely produced "8.0%" as an ungrounded numeric echo (training-data knowledge of Salesforce sample data, or hallucinated from the question framing). No Quote citation was required.
  • Diagnostic detail: In the May 12 CSV, the Control run failed with 3s_args_issue (the connector path didn't actually retrieve the Quote), while the Experiment run was marked Pass with no failure-analysis fields populated. This pattern of "Control fails, Experiment passes with empty diagnostics" is a strong false-positive signal.
  • Fix applied: Assertion now requires both the "8%" value AND a citation of the Quote record 0Q0dM000002JaHxSAK (GenePoint Lab Q2 Quote). Without M8 Quote crawling there is no path to a sourced answer.
